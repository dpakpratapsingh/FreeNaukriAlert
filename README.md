# Free Naukri Alert (Project Tricker)

**Motive:** To serve Competitive Students (with future expansions targeting Government Employees for job upgrades, and international associate branches).
**Platform Vision:** A Non-Profit Publication offering a simple, intelligent notification alert system fueled by strict algorithmic rules.

## Core Rules & Architecture
1. **No Emotional Judgment**: The platform strictly judges *Eligibility*. It does NOT judge personality, actions, or emotions.
2. **Platform Independence**: Designed to run seamlessly across devices with responsive, dynamic themes inspired by the "Indian Competitive Journey" (distraction-free, focused, high-contrast readability).
3. **The 100-Question Assessment**: Evaluates the student's activity percentage by testing the syllabus and the "Examiner Thought Process" (capped at 100 questions).

## System Structure
- **`/01_Documentation`**: Vision, rules, and core workflows.
- **`/02_Platform_Architecture`**: Strategic execution phases (P3 to P6).
- **`/03_SourceCode`**: The core engines.
  - `/frontend`: Next.js Web App (Vanilla CSS, highly thematic).
  - `/backend`: Node.js Engine.

## Professional Setup
- **Maintainability**: Clear, modular codebase separating presentation from business logic.
- **Security**: Built-in boundary checks and robust routing.
- **Backup & Share**: Centralized via GitHub for transparent non-profit collaboration.
